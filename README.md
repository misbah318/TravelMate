# 🌍 TravelMate

TravelMate is a modern full-stack travel booking web application that allows users to explore destinations, view travel packages, check live weather, and manage trip bookings in a smooth and interactive interface.

---

## 🚀 Live Demo

https://your-render-app-url.onrender.com

---

## ✨ Features

- 🏝 Explore travel destinations (India, UAE, Japan, France, etc.)
- 📦 Travel packages with pricing and details
- 📅 Book trips with form validation
- ☁️ Live weather search using OpenWeather API
- 📊 Booking dashboard with:
  - Total bookings
  - Top booked city
  - Top package
- 💾 Backend API for saving bookings
- 🎨 Modern responsive UI
- 🖼 Hero section image slider (auto-changing backgrounds)

# Go inside folder
cd TravelMate

# Install dependencies
npm install

# Run server
